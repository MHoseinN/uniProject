# راهنمای سریع شروع کار

## مراحل اجرا

### 1. نصب MongoDB (اگر نصب نیست)

برای Ubuntu/Debian:
```bash
sudo apt update
sudo apt install -y mongodb
sudo systemctl start mongodb
sudo systemctl enable mongodb
```

برای macOS:
```bash
brew tap mongodb/brew
brew install mongodb-community
brew services start mongodb-community
```

یا استفاده از Docker:
```bash
docker run -d -p 27017:27017 --name mongodb mongo:latest
```

### 2. اجرای پروژه

```bash
# نصب وابستگی‌ها (اگر انجام نشده)
npm install

# اجرا در حالت توسعه
npm run dev
```

### 3. دسترسی به برنامه

باز کردن مرورگر و رفتن به آدرس:
```
http://localhost:3000
```

## تست سریع سیستم

### مرحله 1: ثبت نام مدیر گروه
1. روی "ثبت نام" کلیک کنید
2. تب "مدیر گروه" را انتخاب کنید
3. اطلاعات را وارد کنید:
   - نام: علی
   - نام خانوادگی: احمدی
   - شماره شناسایی: MGR001
   - کد ملی: 1234567890
   - رشته: کامپیوتر
4. ثبت نام کنید

### مرحله 2: ثبت نام استاد راهنما
1. ثبت نام → تب "استاد"
2. اطلاعات:
   - نام: محمد
   - نام خانوادگی: رضایی
   - شماره شناسایی: PROF001
   - کد ملی: 9876543210
   - رشته: کامپیوتر

### مرحله 3: ثبت نام استاد داور
1. ثبت نام → تب "استاد"
2. اطلاعات:
   - نام: حسین
   - نام خانوادگی: کریمی
   - شماره شناسایی: PROF002
   - کد ملی: 5555555555
   - رشته: کامپیوتر

### مرحله 4: ثبت نام دانشجو
1. ثبت نام → تب "دانشجو"
2. اطلاعات:
   - نام: زهرا
   - نام خانوادگی: محمدی
   - شماره دانشجویی: 9812345
   - کد ملی: 1111111111
   - رشته: کامپیوتر

### مرحله 5: تست فرآیند کامل

1. **ورود به عنوان دانشجو:**
   - کد ملی: 1111111111
   - رمز عبور: 9812345
   - درخواست پروژه بدهید

2. **ورود به عنوان مدیر گروه:**
   - کد ملی: 1234567890
   - رمز عبور: MGR001
   - درخواست را تایید کنید
   - استاد راهنما (محمد رضایی) و استاد داور (حسین کریمی) را انتخاب کنید

3. **دوباره ورود به عنوان دانشجو:**
   - موضوعات پیشنهادی بدهید (مثلاً: "سیستم مدیریت آنلاین")

4. **ورود به عنوان استاد راهنما:**
   - کد ملی: 9876543210
   - رمز عبور: PROF001
   - موضوعات پیشنهادی را ببینید و یکی را تایید کنید

5. **دوباره ورود به عنوان دانشجو:**
   - گزارش پیشرفت ارسال کنید

6. **ورود به عنوان استاد داور:**
   - کد ملی: 5555555555
   - رمز عبور: PROF002
   - زمان دفاعیه پیشنهاد بدهید

7. **ورود به عنوان مدیر گروه:**
   - زمان‌های پیشنهادی را تایید کنید
   - سیستم خودکار زمان دفاعیه را به دانشجو اختصاص می‌دهد

8. **ورود نهایی به عنوان دانشجو:**
   - زمان دفاعیه خود را مشاهده کنید

## توجه

- برای تست راحت‌تر، همه کاربران را با رشته یکسان (مثلاً "کامپیوتر") ثبت کنید
- مدیر گروه فقط پروژه‌های رشته خودش را می‌بیند
- استاد راهنما و استاد داور نباید یکی باشند
- هر زمان دفاعیه 20 دقیقه است

```bash
# بررسی وضعیت MongoDB
```

```bash

PORT=3001
```
